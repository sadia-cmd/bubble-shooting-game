# Bubble Shooter Game

A fun arcade-style Bubble Shooter game made using HTML, CSS, and JavaScript.

## Game Description

Bubble Shooter is a colorful browser game where the player shoots bubbles, matches same-colored clusters, clears the board, and tries to score as high as possible.

## How to Play

- Aim at the bubble board.
- Shoot bubbles toward matching colors.
- Match color clusters to pop bubbles.
- Clear more bubbles to increase your score.
- Use fewer shots to perform better.
- Restart the game anytime and try to beat your best score.

## Features

- Arcade-style gameplay
- Score system
- Level tracking
- Shots counter
- Best score display
- Combo, popped, dropped, and row stats
- Restart option
- Clean and interactive game UI
